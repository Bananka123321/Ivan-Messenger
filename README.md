# Асинхронный TCP-чат с шифрованием

Клиент-серверное приложение для защищённого обмена личными сообщениями в режиме реального времени.

## Возможности

- Защищённый обмен сообщениями по протоколу **TLS 1.2+** (OpenSSL)
- Регистрация и авторизация с хешированием паролей (**Argon2**)
- Поиск пользователей по нику с автодополнением
- Личные сообщения с мгновенной доставкой
- История переписки с подгрузкой
- Автоматическое восстановление сессии при разрыве соединения (без повторного ввода пароля)
- Валидация вводимых данных на клиенте
- Кроссплатформенность: **Linux** и **Windows**

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Язык | C++17 |
| GUI-фреймворк | Qt6  |
| Шифрование | OpenSSL (TLS) |
| Протокол | JSON-over-TCP |
| Серверная БД | PostgreSQL |
| Сборка | CMake |

## Скачать

Перейди в [Releases](https://github.com/Bananka123321/async-TCP-server-client/releases) и скачай архив для своей ОС.

### Windows
1. Распакуй `ChatClient_Windows.zip`
2. Запусти `Client.exe`
3. Готово

### Linux
1. Распакуй `ChatClient_Linux.tar.gz`
2. В терминале: `chmod +x run.sh && ./run.sh`
3. Готово

### Подключение
Клиент автоматически подключается к серверу: `31.192.108.37:6767` (TLS)
